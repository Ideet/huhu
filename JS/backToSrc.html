重定向到百度示例

此页面演示了如何使用JavaScript函数 backToSrcLanguage() 实现重定向功能。 点击下方按钮将跳转到 Baidu百度 首页。

JavaScript 代码 
// 定义重定向函数
function backToSrcLanguage() {
    // 执行重定向到百度
    window.location.href = "https://www.baidu.com";
    
    // 也可以使用以下替代方法：
    // window.location.replace("https://www.baidu.com");
    // window.open("https://www.baidu.com", "_blank");
}
注意： 此功能将立即重定向到百度首页。如果您不想离开当前页面，可以右键点击按钮选择"在新标签页中打开"。
